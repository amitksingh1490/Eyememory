# Eyememory
React JS game- eyememory
Environment
NodeJS 6.3.1


Step 1
npm install

Step 2
npm run serve


If you face any issue while npm install check your nodejs version since, I am using node-sass it fails for latest nodejs versions
To switch your node version

1 npm install -g nvm

2 nvm install 6.3.1

3 nvm use 6.3.1

4 nvm current
